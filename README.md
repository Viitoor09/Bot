# 🤖 Telegram Bot - Cotação & Gestão de Usuários

Este projeto é um bot para Telegram desenvolvido em **Python**, focado em automação de consultas financeiras e gerenciamento básico de logs de usuários. Desenvolvido como parte do meu portfólio de estudos em **Análise e Desenvolvimento de Sistemas (ADS)**.

## 🚀 Funcionalidades

- **Cotação em Tempo Real:** Consulta a API de economia para trazer valores atualizados de Dólar e Euro.
- **Inteligência de Texto:** Reconhece palavras-chave em frases (ex: "quanto está o dolar?") sem necessidade de comandos fixos.
- **Persistência de Dados:** Registra automaticamente o nome e ID dos usuários em um arquivo `usuarios.txt` (Log de acessos).
- **Painel Administrativo:** Comando exclusivo para o desenvolvedor visualizar estatísticas de uso.
- **Segurança:** Implementação de variáveis de ambiente (`.env`) para proteção de tokens sensíveis.

## 🛠️ Tecnologias Utilizadas

- [Python 3.13+](https://www.python.org/)
- [pyTelegramBotAPI](https://pypi.org/project/pyTelegramBotAPI/) (Telebot)
- [python-dotenv](https://pypi.org/project/python-dotenv/) (Segurança de dados)
- [Requests](https://pypi.org/project/requests/) (Consumo de APIs externas)
